Notes de design

Icônes
Emojis HTML (🔒 ✅ ⚡)
SVG inline dans un carré coloré — rendu identique sur tous les clients email
Boutons
Border-radius: 8px (au lieu de 6px), font-weight: 600, padding plus généreux
Couleurs par type
● Consentement : bleu (#2563eb)
● Retrait : vert (#16a34a)
● Notification : ambre (#d97706)
● Invitation : violet (#7c3aed)
Compatibilité
SVG inline fonctionne sur Gmail, Apple Mail, Outlook web. Pour Outlook desktop, fallback VML existant reste en place. Les emojis sont simplement remplacés.
Template de base
Structure identique conservée : accent bar gradient + logo S + divider + icon titre + body + footer. Seul le paramètre icon passe de string emoji à SVG string.
Nouveau template
Ajout d'un template « Invitation réunion » avec détails structurés (nom, date, lieu, organisateur) dans un bloc info
